# Ultra-Wideband (UWB) Antenna Design using HFSS

## Overview

This project presents the design and simulation of an Ultra-Wideband (UWB) Antenna using ANSYS HFSS 2025 R2 Student Version. The antenna is designed to achieve wideband performance with good impedance matching, low return loss, and acceptable VSWR over a broad frequency range.

## Software Used

- ANSYS HFSS 2025 R2 Student

## Antenna Specifications

| Parameter | Value |
|------------|------------|
| Antenna Type | Ultra-Wideband (UWB) Antenna |
| Frequency Range | 1 GHz – 15 GHz |
| Resonant Frequency 1 | 3.52 GHz |
| Return Loss (S11) | -10.03 dB |
| Resonant Frequency 2 | 14.51 GHz |
| Return Loss (S11) | -12.49 dB |
| Best Return Loss | ≈ -22 dB @ 10.1 GHz |
| VSWR | < 2 across major operating bands |
